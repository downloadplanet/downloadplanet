# Upstereo Pro
![Upstereo Pro](https://mycommerce.akamaized.net/api/pimages/P300608613/BIG/300608613.JPG)
UpStereo Pro is a stereo enhancement plug-in for audio sequencers and workstations. * Stretches strereo width without using traditional phase techniques. * Unique technology that exaggerates any off centre sounds and pushes them to the stereo edges. * Punch control for enhancing or dampening initial bass or kick sounds. * Sizzle parameter for giving high end transients a shake and sparkle. * Linear equaliser with 16 point spline editing. * Waveshaper with extreme settings. * Full linear processing. * On screen information panels for each of the controls. * 64 bit versions in VST, VST3, AU and AAX formats.
