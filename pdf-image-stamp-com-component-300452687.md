# PDF Image Stamp COM Component
PDF Image Stamp COM Component software automates the process of applying JPEG Images to different sized pdf's
PDF Image stamp contains many special features like starting from any page, 1 in nth pages, x,y positioning on the page, image size scaling, image opacity, flip horizontal page, flip vertical page, alignment (centre, left, right) & list of files processing.
