# Layer2 Taxonomy Manager for SharePoint
Offers a complete taxonomy lifecycle management application to import create deploy improve and export corporate taxonomies using various formats including SKOS Custom term properties classification rules etc

Features Benefits
 Advanced import update features Support of additional formats to import free and commercially available taxonomies eg SKOS Outofthebox SharePoint supports CSV only localization and synonyms are not supported by default
 Export feature Taxonomy Manager allows exporting SharePointbased corporate taxonomies without any loss of information into several different formats eg SKOS Outofthebox SharePoint doesnt offer any export
