# MD5 Password Pro
MD Password is a password recovery tool for security professionals which can be used to recover a password if its MD hash is known MD is an industry standard hash algorithm that is used in many applications to store passwords MD Password features fast highly optimized recovery engine that can recover multiple hashes at once up to hashes in the Pro version The number of hashes being recovered simultaneously does not have significant performance impact The recovery engine is fully configurable so you can adjust the recovery options as needed
Since there is no known way to reverse the MD algorithm MD Password provides a number of tools to recover the password
Brute Force Attack
Dictionary Attack
Hybrid Dictionary Attack
Smart Force Attack
