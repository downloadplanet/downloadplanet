# ZylTimer (Delphi/C++Builder) - Single Developer License
ZylTimer is a high resolution longterm Delphi CBuilder timer component which provides a higher precision than the standard Delphi C Builder TTimer component TTimer component which ships with Delphi CBuilder uses the Windows Message Queue to generate the OnTimer event Due to this approach its impossible to get accurate timer intervals smaller than milliseconds ZylTimer is a thread based timer and due to this architecture provides a real precision of millisecond which is inevitable in time critical applications The OnTimer event is always fired in time which is not available for the standard TTimer when there are too many messages processed The standard TTimer component uses the SetTimer API function so the interval is limited to milliseconds about days ZylTimer is a thread based timer and the interval could be set even to hundred years if necessary
