# Modern PDF Server
Unlike many similar server-based software, Modern PDF Server intercepts documents extremely fast on their way to the printer, creating a version in Adobe's PDF format instead of spitting out a physical piece of paper.

Users can choose the page size and orientation, resolution, and whether a font should be embedded or not. It allows documents to be encrypted, and user permissions to be set.
