# Word Password Recovery (for personal use)
Word Password Recovery is a program to recover lost or forgotten passwords to the Microsoft Word documents protected with password to open. The program supports the "brute-force" attack, dictionary-based attack and dramatically fastest "Booost-Up" attack. The program automatically saves password searching state and can resume an interrupted attack (after a stop, crash, etc.)
