# SolveigMM WMP Trimmer Plugin 3 - Home Edition
![SolveigMM WMP Trimmer Plugin 3 - Home Edition](https://mycommerce.akamaized.net/api/pimages/P300165648/BIG/300165648.JPG)
SolveigMM WMP Trimmer Plugin 3 - Home Edition is a fast and small tool to losslessly cut or edit out any part from AVI, WMV, ASF, MP3,MPEG-2* and WMA files loaded to Windows Media Player. No quality degradation. No out of sync problems. No AVI size restrictions.


* Based on SolveigMM Video Editing Engine
* Supports any AVI Format files - (*.avi )
  o DV AVI type 1, 2; OpenDML
  o Any video content. DivX; XviD; 3ivX, etc.
  o Any audio content. MPEG-1,2 Layer I, II, III; AC3; OGG, etc.
  o  VBR MPEG audio. Keeps the synchronization
  o Large AVI files. More than 2 and 4 GB

* Supports Windows Media Format files - (*.asf, *.wma, *.wmv, *.wm)
  o Any video content. WMV 1,2,3; MSS2;  MPEG-4 AVC,  etc.
  o Any audio content. WMAudio V 2,7,8; MPEG-1,2 Layer I, II, III; AC3, etc.
 
 * Supports MPEG Audio Format files ( *.mp1, *.mp2, *.mp3, *.mpa )
  o MPEG-1 Layer I, II, III
  o MPEG-2 Layer I, II, III
  o Supports MPEG-2 files - (.mpg, .vob, .ts) (* requires SolveigMM Video Splitter installed)
* Key frame/GOP accuracy
