# Picture-in-Picture Presentation
OfficeOne Picture-in-Picture Presentation adds new features to PowerPoint that allows you to add next slide thumbnail to each slide in your presentation. You can change the position and size of each thumbnail on the slide. Click the Insert button again updates the thumbnails with slide updates. It also adds a "delete" button allows you to delete all next slide thumbnails on all slides.
