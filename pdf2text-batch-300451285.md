# PDF2Text Batch
PDF2Text Batch is a plugin conversion tool for Acrobat Used to convert your PDF documents into plain text files. Options are available to output page numbers plus text tolorance level for varing sizes of text. This Batch version can process batch lists of many pdf files. NOTE: requires full version of Acrobat to operate correctly.
