# SMWordDocument for Delphi/C++Builder
SMWord suite for Delphi/CBuilder is a native VCL components for direct and fast text extraction from any MS Word document. MSWordDocument component allow to read any Word document directly without OLE automation of MS Word application (MS Word installed is not required). Direct text extraction allow to process large documents very-very fast. Summary Information is available too. You may use this component to provide a search engine - for example, find document where some substring is occured or to display summary information and text preview in own file manager.
