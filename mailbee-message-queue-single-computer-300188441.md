# MailBee Message Queue (Single Computer)
MailBee Message Queue is a queue-based email-sending system which operates as a Windows NT service intercepting "send e-mail" requests from MailBee.SMTP objects and then sends email in a background. This allows MailBee.SMTP object to operate very quickly because MailBee.SMTP object is no longer waits for the email message to be actually sent. It is especially important when your application or ASP web page sends multiple messages at once.
