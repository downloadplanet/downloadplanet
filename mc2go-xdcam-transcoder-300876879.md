# MC2GO XDCAM Transcoder
MainConceptⓇ 2GO XDCAM Transcoder is an optimized Docker container for file-based transcoding of your media files into professional Sony camera formats like XDCAM HD, XDCAM EX, XDCAM IMX and DVCAM (XDCAM DV), incl. REST API control.

The demo version of the MC2GO XDCAM Transcoder is fully functional. It only has an audio encoding limitation of recurring silence, i.e. after some time audio will be muted and continues again, etc.

MC2GO XDCAM Transcoder only works when the Docker environment is installed properly on the system where you want to run the MainConcept microservice module on (not included).
