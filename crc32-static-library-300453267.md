# CRC32 Static Library
CRC Static Library for Microsoft VC was produced to make your CRC checking operations quick easy to implement seamlessly without the need for an external DLL The algorithm of CRC is exactly the same as that internally used by PKZIP and WinZip ie the CRCs calculated by PKZIP and CRC are identical That’s another fine thing about CRC it calculates compatible widely used bit checksums not just any numbers You can use the pkunzipexe vt filenamezip command to see that PKZIP’s CRC values of files stored inside the filenamezip are identical to that calculated by CRC for these files
