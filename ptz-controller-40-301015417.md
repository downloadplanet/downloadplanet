# PTZ Controller 4.0
PTZ Controller software is to control Pan Tilt Zoom cameras.
It is a virtual keyboard for PTZ camera. It can control PTZ camera through serial port and network interface. It supports multiple protocols. User-defined functions and keyboard shortcuts make it more powerful than traditional PTZ keyboard hardware. It's also a good debug tool for testing and install of PTZ camera.
