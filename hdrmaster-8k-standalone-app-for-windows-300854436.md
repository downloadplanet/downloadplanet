# HDRmaster 8K (Standalone App for Windows)
HDRmaster allows you to edit Metadata related to HDR in H.265/HEVC streams. It's also a signal analysis tool where you're able to read raw signal levels of transport streams or containers. You'll be able to see raw YcbCr values as well as decoded RGB values and nit levels. HDRmaster is capable of exporting 16 Bit TIFF-Files from 8Bit or 10Bit videos. If you're doing Prostproduction work with High Dynamic Range video or if you desire to analyze signal levels - HDRmaster is the tool of your choice. Colorists can export still frames from any given HDR video and make sure they're grading new contents accordingly.
