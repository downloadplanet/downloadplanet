# UserMonitor (3 computer)

Ideal for teacher, instructor or administrators.
2. Monitoring
UserMonitor shows you what the other network computer users see on their display screen. All used programs are also listed (with path detail) in real time. Including History function.
1. Demonstrating
UserMonitor is brilliant for the lesson as well as trainings at the computer! Educationalists can with this software transfer their display screen to all pupil computers screens to demonstrate something.
3. Helping
You can write messages or close unwanted programs to any computers in the network.
