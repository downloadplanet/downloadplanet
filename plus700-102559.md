# PLUS700
PLUS700 is a Checkers game for Windows. It uses the Standard Laws of American Checkers. Plus700 is a self-teaching checkersplayer. That is, the more you play against Plus700, the more difficult it becomes to beat. Options: - self-teaching; - "ponder" mode; - any starting position on the board; - save/load position/game; - position analysis; - function "undo/repeat moves"; - function "rotate board"; - function "best next"; - recording game protocol; - different play opponent: computer, another player or PLUS700 play herself; - multilanguage support; - import/export Database file.
