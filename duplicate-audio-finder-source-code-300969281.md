# Duplicate Audio Finder Source Code
![Duplicate Audio Finder Source Code](https://mycommerce.akamaized.net/api/pimages/P300969281/BIG/300969281.PNG)
Duplicate Audio Finder is a utility for scanning your computer and finding duplicate copies of your audio files. Duplicate Audio Finder will find audio files that are an exact copy of each other but also can compare audio files by content and give a match for audio files that have different resolutions, bit rate or format but contain the same audio.

Features:
Find duplicate audio files that are exact copy of each other
Find duplicate audio files that have the same content
Matching audio files are displayed in groups
Preview matching audio files and their ID3v1, ID3v2, APE Tag and Vorbis comments information and album cover pictures (BMP, JPEG, PNG, GIF)
Delete audio files that are not needed
Multi-threaded processing
Full unicode support
