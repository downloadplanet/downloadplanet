# StationPlaylist Streamer
Stream audio from the soundcard line input to an Icecast or Shoutcast compatiblke server, using MP3, HE-AAC (AAC+), or Ogg Vorbis codecs.
Provides optional audio processing with a compressor/limiter DSP.
Supports retrieving artist & title details for metadata by polling a text file created by StationPlaylist Studio or other compatible software.
