# True Slide Resize
OfficeOne True Slide Resize allows you to change the size of your PowerPoint slides to match the screen size. With True Slide Resize, you can now change the aspect ratio between 4:3, 16:9 and 16:10 quickly without distorting your slide content.
In PowerPoint, you can change the size of your slides through the Page Setup dialog box. This makes PowerPoint resize the slides but it also results in stretching of various objects on the slide. OfficeOne True Slide Resize allows you to change the size of your slides without distorting the shapes on the slide. It handles various shapes on the slide as well as the slide masters to provide a complete solution.
OfficeOne True Slide Resize works with Microsoft PowerPoint 2010 (32-bit as well as 64-bit), Microsoft Office PowerPoint 2007, Microsoft Office PowerPoint 2003, Microsoft PowerPoint 2002 (XP) and Microsoft PowerPoint 2000. Download and install True Slide Resize - it will automatically register with your PowerPoint environment. You can access True Slide Resize on the Home tab (PowerPoint 2007 and later versions) or the True Slide Resize menu item in the Edit menu (PowerPoint 2003 and earlier versions).
