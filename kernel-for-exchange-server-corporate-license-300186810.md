# Kernel for Exchange Server - Corporate License
![Kernel for Exchange Server - Corporate License](https://mycommerce.akamaized.net/api/pimages/P300186810/BIG/300186810.PNG)
Kernel for Exchange Server is EDB repair software which repairs the EDB and STM files and retrieves the inaccessible user mailboxes. User mailboxes and emails which have been permanently deleted from the EDB database are recovered by the software and displayed in Red color. Exchange Dirty Shutdown and Checksum errors are efficiently handled by the software and recovered mailboxes are saved to PST , Live Exchange ( On - Premise ) and Office 365 mailboxes. Software provides three recovery modes: Automatic Analyze and Recover, Advanced Scan and Rebuild Corrupted Database. With these modes users can repair the EDB database and have the mailbox up and working again on MS Exchange Server. Emails, contacts, calendars, images, appointments, alarms, drafts, deleted emails, Unicode characters, journals, email properties are recovered successfully. Kernel for Exchange Server supports database repairing and email recovery from EDB databases created with MS Exchange Server 5.0, 5.5, 2000, 2003, 2007, 2010, 2013 and 2016. Users can download the Free evaluation version of the software and experience the working process of the software. The demo software will display preview of mailbox emails and other items and save 25 items from each folder.
