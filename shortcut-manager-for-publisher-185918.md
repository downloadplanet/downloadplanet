# Shortcut Manager for Publisher
Shortcut Manager for Publisher enables you to
Define your own keyboard shortcuts to menu items and recorded macros
Define shortcuts to commands that arent available directly through the Publisher User Interface
Shortcut Manager for Publisher works with Microsoft Office Publisher Microsoft Office Publisher as well as Microsoft Publisher Download the addin and install it in your machine After the addin is loaded successfully you can start assigning shortcuts using the customize shortcuts dialog The customize shortcuts dialog can be invoked by selecting Tools Customize Shortcuts
