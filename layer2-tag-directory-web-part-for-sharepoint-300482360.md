# Layer2 Tag Directory Web Part for SharePoint
Renders the Microsoft SharePoint 2010 / 2013 managed metadata taxonomy tree as flat A-Z directory category index directly for content discovery and navigation. Flexible link building, e.g. link to Tag Profile page or content result list. Optionally additional information, e.g. tag description and usage count. Template-based design.
