# Vovsoft Copy Files Into Multiple Folders
Copying files into multiple directories is very easy using Copy Files Into Multiple Folders. All you need to do is to choose the files and choose the directories. Then pressing the Copy Files to Folders button starts the process. You can drag and drop the files and folders. You can select to overwrite or not to overwrite the existing files.
