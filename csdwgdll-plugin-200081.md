# CS_DWG.DLL plugin
CS_DWG.DLL Plugin for IrfanView and ABC-View Manager
CS_DWG.DLL Plugin provides support for AutoCAD DWG files in IrfanView, XNView and ABC-View Manager.
Features:
Fast reading
3D graphic
Changeable image size
