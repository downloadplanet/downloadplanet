# K-FTP
K-FTP is an Active-X componenent for Microsoft Visual Basic 6. Thanks to K-FTP you can very easily download or upload files to a FTP server from your VB6 application. This only requires a few lines of code. K-FTP's developper's package contains an help file and a sample application : complete FTP client source code.
