# Virtual Serial Port Kit
![Virtual Serial Port Kit](https://mycommerce.akamaized.net/api/pimages/P196880/BIG/196880.GIF)
Virtual Serial Port Kit creates virtual serial ports and connects each pair of them via a virtual nullmodem cable
All the data written to one virtual serial port can be immediately read from the other one and vice versa
So your applications can exchange data just like they do via hardware serial ports and real nullmodem cables
