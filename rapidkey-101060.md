# RapidKey
![RapidKey](https://mycommerce.akamaized.net/api/pimages/P101060/BIG/101060.JPG)
Rapid Key provides you with a new Windows functionality:
After starting RapidKey you find the RapidKey icon on the taskbar. Then in all Windows applications you have access to:
- Autotexts
- macros (keystroke, mouse clicks + movements)
- desktop
- favorite folders
