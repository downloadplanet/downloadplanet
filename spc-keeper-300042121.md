# SPC Keeper
SPC Keeper is a simple, affordable data repository for all types of attribute data from the factory floor. It is easily configured for part or process based data collection. Validate data against control limits and spec limits. Optionally, review simple Statistical Process Control charts for real-time SPC. Link a high powered SPC analysis package to SPC Keeper using standard ODBC links or export the data with a click of a button to Excel. Monitor and measure your product to ensure conformity.
