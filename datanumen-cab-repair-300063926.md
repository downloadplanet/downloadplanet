# DataNumen CAB Repair
DataNumen CAB RepairDCABR formerly Advanced CAB Repair is a powerful tool to repair corrupt CAB files It can scan the damaged CAB archives and recover your data in them as much as possible so to minimize the loss in file corruption

Main features
Support to fix all version of Microsoft cabinet files
Support MSZIP LZX and Quantum compression methods
Support to repair CAB files on corrupted medias such as floppy disks Zip disks CDROMs etc
Support to repair a batch of CAB files
Provide an EXE stub and output a SFX file when necessary
Integrated with Windows shell so you can repair a file with context menu in the Explorer easily
Support to find and select the CAB files to be repaired on the client computer
Support drag drop operation
Support command line parameters
