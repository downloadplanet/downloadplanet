# Mac PDF Measure It
Mac PDF Measure It is an Acrobat plugin tool, it enables accurate measurement of the distance and angle between two points on a page. To measure a distance, click anywhere on a PDF page to set a reference point, then click again in the same page to set a measure point, Measurments include horizontal, vertical, distance and angle degrees, changeable colored crosshairs are placed over the page at both points. Measurements are shown in either points, picas,inches, feet, yards, centimeters, millimeters & meters. Requires full version of Acrobat
