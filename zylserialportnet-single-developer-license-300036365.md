# ZylSerialPort.NET - Single Developer License
ZylSerialPort.NET is a thread based, event driven, asynchronous serial port .NET component (class library). Use ZylSerialPort.NET component to easily communicate with external devices on serial port connection, such as modems, bar code readers, GSM modules and others. ZylSerialPort.NET is written 100% in C# using .Net framework and API calls.
