# ProjectArchitect
ProjectArchitect is a Project Manager that have been designed to improve efficiency when dealing with multiple project / multiple tasks leaders at the same time. Thanks to ProjectArchitect, you'll be able to assign tasks leaders to one or more projects and to affect specific tasks to each task leader. ProjectArchitect will also make to also to manage versions and deadlines. It will also generate ChangeLogs, sends them by FTP or email.
