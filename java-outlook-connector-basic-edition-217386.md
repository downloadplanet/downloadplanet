# Java Outlook Connector - Basic edition
Intuitive and easy to use library to integrate your Java desktop application with Microsoft Outlook.
Supported Outlook items are:
Contacts
E-mails
Tasks
Appointments
Journal items
Notes
Attachments
Folders

Based on a robust model, Java Outlook Connector handle automatically native resources memory allocation and COM multi-threading issues.

Compatible with Outlook 2000, 2002, 2003, XP, 2007 and 2010.
