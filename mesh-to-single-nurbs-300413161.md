# Mesh to single NURBS
![Mesh to single NURBS](https://mycommerce.akamaized.net/api/pimages/P300413161/BIG/300413161.JPG)
Mesh to single NURBS is a standalone application This software can convert an open mesh to a single NURBS surface just by clicking one button The feature includes
 Import stl file
 Convert mesh to a single NURBS surface
 Export the surface to igs file
 Controllable precision The maximum deviation between mesh and surface can be specified
 Controllable smoothness The surfaces smoothness can be adjusted by levels
 Handle large scale data Surface control point number can reach X which allows to fit large scale data
 Easy to use Just click one button to create surface from mesh
