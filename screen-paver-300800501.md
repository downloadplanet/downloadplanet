# Screen Paver
![Screen Paver](https://mycommerce.akamaized.net/api/pimages/P300800501/BIG/300800501.PNG)

The program will display an unlimited number of your own jpeg, png, or bitmap pictures from local and/or network directories. Features include picture sorting, stretching/shrinking/tiling, transition effects, picture sets, picture preview, favorite picture saving, wallpaper creation, long filename captions with date, pause/continue control, and more.
