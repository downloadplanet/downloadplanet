# Network Recycle Bin Tool - single copy
The Network Recycle Bin Tool allows you to recover deleted files. It moves deleted files from network drives to the recycle bin. Unlike backup software you can restore a deleted file as it was at the time of deletion. You can define the list of network drives or network folders to track deleted files. Also you can specify the max file size to limit a number of tracked files. Additionaly it offers you the Undelete Tool which prevents deletion of network files for specified folders according the file mask. Export and import functions help you to install software with same options on network machines. The password control disallows unauthorized access.
