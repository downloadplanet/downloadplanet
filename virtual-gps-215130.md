# Virtual GPS
Virtual GPS is a GPS simulator software. You can use it to simulate a GPS receiver unit connected to your system. It's ideal when you need GPS input, but you have no signal or for development purposes. It supports various NMEA sentences, so it can be used with any GPS mapping software. You can use it using a null-modem cable connected to two physical serial ports or you can use one of the virtual serial port software from the market.
