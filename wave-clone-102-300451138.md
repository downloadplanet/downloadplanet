# Wave Clone 1.02
Wave Clone is a Windows multimedia tool allowing you to share Wave (MME) ports among several applications. Because many audio (wave) drivers are single-client, allowing only single application to use each port at the same time, you cannot, for example, record from the microphone to two applications simultaneously. Wave Clone brings multi-client feature to any audio port in your system allowing unlimited number of applications to use them at the same time.
