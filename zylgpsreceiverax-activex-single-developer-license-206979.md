ZylGPSReceiverAX is an ActiveX control that communicates with a serial GPS receiver. This control works with any NMEA compatible receiver connected to one of the serial ports. You can use it also with USB devices, because these devices usually have a driver that redirects the input from the USB port to a virtual serial port. With this control you will be able to develop robust GPS applications in any ActiveX supporting developement environment. Be one of the hundred developers who use this powerful solution.
