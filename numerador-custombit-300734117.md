# Numerador Custombit
Numerador is a Windows Visual Numbering Software for printing numbered forms invoices vouchers raffles tickets or any printed work with sequential numbers on it Your definitive bates numbering software

Each numerator is represented by a numbering label which is located on the paper sheet visually dragging it with the mouse or keyboard arrows to its actual location

The editor allows you to select a background image which serves as a graphical reference and can then be printed or not depending on whether the numbers must be printed on preprinted forms for instance
