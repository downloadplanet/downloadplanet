# PDF Summary Maker
PDF Summary Maker allows you to change the summary information in many pdf's or individual pdfs with ease, this software also includes summary reports. you can change the Creator, Producer, Creation Date, Author, Title, Subject and Keywords fields within pdf's. Database support through a DSN connection for retrieving and updating NOTE: this software requires Acrobat 5 or greater std/pro. (note: 7.0.0-7.0.3 has an update summary bug in Acrobat)
