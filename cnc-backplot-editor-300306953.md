# CNC Backplot Editor
Let this program draw the tool path of any GCode program on the screen This program will help you write NC code programs or to learn how to write NC programs or event to debug such programs Our software supports the standard G Code commands and also sub program calls and macro programming with IF statements and WHILE loops etc The program draws a graphical representation of the programmed GCode tool path in an isometric D view You can configure the program to interpret the GCode as Mill or Lathe Format A or B codes Your GCode program will be shown with syntax highlighting in the editor With the code editor you can open edit and save GCode programs up to a size of GB very fast The reason is that we have build in our Text Editor Lighting fast technologie into this program The editor supports an unlimited number of undo and redo steps All inputs can be converted into capital letters automatically You can also convert all characters of existing files into capital letters Import DXF files and convert DXF files into Gcode Or export the tool path of your NC program as DXF file You can use this program to find errors in the GCode If you click on a line in the D view the line of GCode will be selected in the editor And if you go to a line in the editor the end point of the tool movement with be selected in the D view You can run the program line by line you can press the pause button and you can let the program run slower You can see the current values of all variables if you activate the special tool window with CalculateShow variables If you are in the pause modus you can calculate any expression like SIN for example for the variable values of the actual line of code with the function CalculateRender expression With the DNC function you can transfer your programs to your CNC machine
