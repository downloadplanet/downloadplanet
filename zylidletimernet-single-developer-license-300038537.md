# ZylIdleTimer.NET - Single Developer License
ZylIdleTimer.NET is .NET component (class library) which helps you to take actions after a specified system-wide (related to the whole system) idle time or to check the time interval of user inactivity. System Idle time is the time interval of full user inactivity. In this interval any user action is missing on the computer, like pressing a key of the keyboard or moving the mouse or pressing a mouse button or rotating the scroll. ZylIdleTimer.NET is written 100% in C# using .Net framework and API calls.
