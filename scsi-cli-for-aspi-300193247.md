# SCSI CLI for ASPI
The SCSI CLI for ASPI software package allows user to execute the SCSI commands via the ASPI. All executable modules are implemented as a console applications that accept the number of arguments in the command line. The command line arguments allow user to specify the SCSI device and the SCSI command parameters. It is possible to rescan the SCSI bus of the host bus adapter and to enumerate the SCSI devices. Optimized for Tape Drive and Media Changer devices.
