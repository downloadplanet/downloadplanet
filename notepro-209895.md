# NotePro
![NotePro](https://mycommerce.akamaized.net/api/pimages/P209895/BIG/209895.JPG)
NotePro is a highly capable rich-text editor and word processor. It offers nearly all of the features found in standart Windows WordPad but adds spell checking, URL detection, a calculator, and bookmarking functions. NotePro also uses an MDI approach, so you can work with several documents at once within a single main window. Other features include a toolbar, formatting bar, and status bar object and picture insertion and several appearance and behavior options.
