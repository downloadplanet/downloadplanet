# Tareas MSD
![Tareas MSD](https://mycommerce.akamaized.net/api/pimages/P300381979/BIG/300381979.JPG)
Tareas MSD es un planificador de tareas para múltiples trabajadores que funciona en ordenadores con Windows Tareas MSD se basa en dos conceptos básicos Equipos de trabajo este programa está pensado para gestionar los trabajos de varias personas aisladamente u organizadas en equipos de trabajo Por esta razón la unidad básica de funcionamiento del programa es el Equipo de trabajo que puede estar compuesto por una o más personas Tareas las actividades que desarrollan los equipos de trabajo se guardan en el programa en registros llamados Tareas que pueden ser citas visitas trabajos manuales reuniones etc Es muy fácil de utilizar y permite configurar muchas de sus características para adaptarse a los gustos del usuario Permite obtener la información filtrada por condiciones sencillas y potentes así como buscar texto por cualquier campo Todos los datos se pueden obtener por pantalla y por impresora Cada tarea lleva asociado de forma automática un número de orden de trabajo que puede imprimirse para que la firme el cliente Se pueden utilizar los distintos estados de las tareas para realizar un seguimiento del cumplimiento de las mismas pendiente iniciada terminada suspendida y anulada Incluye un módulo estadístico que permite conocer en detalle el grado de cumplimiento de las tareas de cada equipo de trabajo Las tareas se pueden asociar a Proyectos y a Clientes Esta información se puede utilizar para encontrar fácilmente las tareas que pertenecen a un proyecto o a un cliente determinado Todos los informes que genera el programa se pueden grabar en disco en formato PDF Estos son algunos ejemplos de las actividades que se pueden gestionar con este programa Una secretaria podría gestionar la agenda de uno o más jefes controlando las visitas reuniones trabajos viajes etc Un encargado podría gestionar las actividades de sus empleados con la posibilidad de tener almacenados trabajos predefinidos que se suelen realizar rutinariamente Una empresa de servicios podría gestionar los trabajos de sus empleados con la posibilidad de clasificarlos por proyecto y asignarlos a un cliente determinado Un profesional podría gestionar las visitas de sus clientes asignándoles día y hora para las visitas
