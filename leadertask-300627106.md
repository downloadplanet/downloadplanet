# LeaderTask
LeaderTask is a userfriendly task management app
Create your lists of tasks for every day arrange them by projects divide tasks into subtasks set reminders for important tasks attach documents to tasks assign them to your friendscolleagues and a lot more
What does LeaderTask can offer you but other cant
✰ You can divide tasks into subtasks in any number and nesting level Subtasks can be also divided into smaller tasks
✰ Unique system “Antiforgetfulness” It allows you to remember all the tasks you planned to perform yesterday and days before yesterday but you didnt do this The system transfers overdue tasks in the list of tasks for today
✰ Real offline mode you can work with tasks even without the Internet connection
✰ You can visually highlight tasks by coloring them
✰ All time management systems are supported GTD Autofocus Superfocus Pomodoro Technique Do It Tomorrow Stephen Covey System and so on
Users of the program has already created more than tasks
