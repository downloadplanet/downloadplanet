# TRegistration Shareware License
![TRegistration Shareware License](https://mycommerce.akamaized.net/api/pimages/P300091298/BIG/300091298.PNG)
TRegistration is Delphi component for use in Win32 (XP/Vista/7/8/10) and Win64 software.
Aimed for shareware developers for handling the registered/unregistered state of applications and managing the registration process and database.
Part of the component is the Registration Tool, with which you can handle and manage the MySQL registration database (comes with full source code).
The system is designed for manual and/or automatic use
Registering a user takes a couple of seconds (there is built-in support for share-it order e-mails also which makes it's usage really simple)
Multiple projects can be managed
Automatic processing and sending of registration number e-mails
Full unicode support
For more info visit: www.3delite.hu
