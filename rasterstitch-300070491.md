# RasterStitch
RasterStitch is a utility for putting together multiple raster images stitching Use this program to create a panorama of several photographs or scan and join fragments of a paper document which dimensions exceed the scanner workspace For example you can scan a draft of the A format with a desktop A x cm scanner or scan a map case of x cm by individual fragments and then join them to create the whole picture The stitch line is created by setting two pairs of common points on the next images That allows even a newbie to accomplish a great result quickly and easily
