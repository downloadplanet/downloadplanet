# RS232 logger ActiveX
RS232 logger ActiveX is an ActiveX component that uses Advanced Serial Data Logger to get either data from an RS232 or COM port or already processed and converted data besides the standard export tools in Advanced Serial Data Logger.

RS232 logger ActiveX can be used to extend functionality of Advanced Serial Data Logger and/or as one more integration method of Advanced Serial Data Logger to your hardware or software application. It is an optimal way for reading data from an RS232 or COM port to Microsoft Office, MatLab and LabVIEW.
