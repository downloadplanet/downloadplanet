# Serial Port Mapper
![Serial Port Mapper](https://mycommerce.akamaized.net/api/pimages/P219751/BIG/219751.GIF)
Serial Port Mapper allows mapping any serial port to any new port name. In other words, it allows creating aliases for serial ports.
For example, there is COM7 port but your favorite software can work with COM1...COM4 ports only. You can assign COM2 alias for COM7 port. You software can open COM2 port but actually COM7 port will be opened.
