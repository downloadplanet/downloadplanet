# Serial Port Mapper
![Serial Port Mapper](https://mycommerce.akamaized.net/api/pimages/P219751/BIG/219751.GIF)
Serial Port Mapper allows mapping any serial port to any new port name In other words it allows creating aliases for serial ports
For example there is COM port but your favorite software can work with COMCOM ports only You can assign COM alias for COM port You software can open COM port but actually COM port will be opened
