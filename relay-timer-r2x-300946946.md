# Relay Timer R2X
Relay Timer R2X is programmable timer software for setting timing features of the relay. It works with 2-channel relay controller boards of National Control Devices (NCD) . It supports Com Port and Network connection between PC and relay board. The connection status is detected and shown continuously. It will remind after disconnection and auto recover after reconnection. Relay can be turned on/off manually and automatically according to powerful schedule setting. The current status of relay is updated and displayed simultaneously.It supports remote access. It can be shown on status bar when the program is minimized. The program can be set to run automatically after login Windows. It can also be launched by command line with specified configuration file or schedule setting file.
