SMExport for Delphi/CBuilder is a component suite for data export from TDBGrid/TDataSet (with BDE or without) into: - XLS (Excel spreadsheet without OLE) - MS Excel (using OLE) - MS Word (using OLE) - MS Access (using DAO 3.5 or 3.6) - Text (fixed or CSV) - HTML - XML - Lotus 1-2-3 - QuattroPro - RTF (Rich Text format) - SYLK (Symbolic Link) - DIF (Data Interchange Format) - SQL-script - Paradox/DBase/ASCII files - MS Windows clipboard - any ADO data source - other dataset component.
