# Word-to-LaTeX (Professional)
Word-to-LaTeX converts Microsoft Word documents to LaTeX and XML formats (additional markup formats can be easily added). Word-to-LaTeX converts all the document contents including tables, lists, images, equations, various styles and formatting, footnotes, bookmarks, etc. The conversion process is fast, accurate, and highly configurable. Word-to-LaTeX can be used as a Microsoft Word add-in, a standalone graphical application or a command-line tool.
