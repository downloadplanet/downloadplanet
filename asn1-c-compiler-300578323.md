# ASN.1 C Compiler
ASN C Compiler can automatically translate ASN specifications into C data structures plus metadata for encoding and decoding ASN messages using one of the encoding rules BER CER DER PER Developers can easily integrate these codes into their applications In addition to the ASN C Compiler ASN C Runtime Library is also part of the package The ASN C Runtime Library is delivered as a set of lowlevel primitive C routines for encoding and decoding the base ASN types BOOLEAN INTEGER etc and a rich collection of utility functions to operate on application messages The ASN C Runtime Library is ANSI C compliant so that it can run on a widevariety of platforms and processors from mainframes to embedded systems
