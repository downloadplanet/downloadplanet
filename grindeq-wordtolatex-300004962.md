# GrindEQ Word-to-LaTeX
![GrindEQ Word-to-LaTeX](https://mycommerce.akamaized.net/api/pimages/P300004962/BIG/300004962.PNG)
This module converts Microsoft Word documents to LaTeX, AMS-LaTeX, Plain TeX, or AMS-TeX format. Microsoft Equation, old Equation Editor 3.x, and MathType are supported. Works with Microsoft Word for Windows.
GrindEQ Math Utilities include modules:
Word-to-LaTeX
LaTeX-to-Word
MathType-to-Equation
