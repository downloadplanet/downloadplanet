# Point cloud to NURBS
![Point cloud to NURBS](https://mycommerce.akamaized.net/api/pimages/P300416869/BIG/300416869.JPG)
Point cloud to NURBS is a standalone application This software can fit a NURBS surface to a point cloud by clicking one button The features include
 Import point cloud file formatted by xyz and txt
 Fit point cloud to a NURBS surface
 Export the surface to igs file
 Controllable precision The maximum deviation between mesh and surface can be specified
 Controllable smoothness The surfaces smoothness can be adjusted by levels
 Handle large scale data Surface control point number can reach X which allows to fit large scale data
 Easy to use Just click one button to create surface from point cloud
