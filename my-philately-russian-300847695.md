# My Philately (Russian)
![My Philately (Russian)](https://mycommerce.akamaized.net/api/pimages/P300847695/BIG/300847695.JPG)
My philately was created to help philatelists with organizing and managing an electronic catalogue of their collection

Basic features of the program
Adding information on unlimited amount of collection objects
Splitting the catalogue into collections and groups
Handling informational characteristics of objects with list creation ability
Picking objects depending on their characteristics
Attaching files or web links to the objects
Adding images viewing and previewing are possible
Handling periodic data on prices
Adding information on unlimited amounts of printed catalogues of stamps
Quick keyboard search and advanced search in the collections
Multiplelevel sorting
Creating reports with the ability to save as MS Excel and MS Word documents
Importing data from text files and MS Excel files
Collection archiving
