# K-Mail ActiveX
K-Mail is an Active-X componenent for Microsoft Visual Basic 6. Thanks to K-Mail you can very easily send emails from your VB6 application. K-Mail support HTML messages with images and sound, multiple attachments, worldwide characters sets, priority levels... This only requires a few lines of code. K-Mail's developper's package contains an help file and a sample application : complete SMTP (email) client source code.
