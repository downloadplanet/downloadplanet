# MC Audio Tools
The MC Audio Tools is an app for Windows desktop which allows you to do many operations on audio (and video) files, fast and easily. You can convert, join, split and cut multimedia files, record audio from any source in your PC, set the ID3 tags of many types of multimedia files - all from the same and simple form with a few mouse of clicks only.
