# Visual MODBUS Explorer
Visual MODBUS Explorer application runs on the Windows operating system platforms and is intended for advanced MODBUS device testing. Application visualizes the host MODBUS hardware configuration and allows the user to work with the multiple MODBUS devices at the same time. Application allows the user to manually execute the MODBUS commands with the different parameters and data. Application visualizes the process of MODBUS command execution, logs the command parameters, the command completion status, the outbound and inbound command data and parses them.
