# PDF Page Size Split Batch
PDF Size Split Batch is an Acrobat plugin that splits a pdf into many pdfs on page sizes within the source pdf this is very useful for spliting mixed pdfs into printable documents on size eg if source pdf had pages of A then Pages of A the output would be sourceApdf sourceApdf containing those pages for that size Batch version allows you to batch process multiple files from a file list
