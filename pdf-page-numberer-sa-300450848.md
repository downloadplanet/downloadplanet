# PDF Page Numberer SA
PDF Page Numberer SA is for numbering pdf files with a few mouse clicks without the need for Adobe Acrobat, you can put customizable page numbers anywhere on the page, with any font size, font name & font type also, features include saving settings for later use for other pdf's, page start & end ranges, start and stop page counts for page groups, watermarking/opacity text, roman numerals, bates numbering, RGB or CMYK text stamping, filename parts, date stamping, file created/modified time stamping, centre on variable size pages, angled text support, retains PDF file version for printing systems, increment group page numbering e.g. page 1, page 1, page 2, page 2 etc, variable dynamic page numbering by sequence e.g. a,b,c,d,e,f etc, preview text, tooltips + many other options.
NOTE: SA version is standalone i.e. Adobe Acrobat is NOT required.
