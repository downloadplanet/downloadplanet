# InkTools
InkTools extends PowerPoint and adds touchfriendly ink tools for use during the slide show Using those tools you can do the following
Add a blank slide after the current slide for recording feedback explain presentation in more detail among its uses
Switch between Arrow pointer and Pen pointer
Select pen colors at the touch of a button
InkTools works with Microsoft PowerPoint bit as well as bit Microsoft Office PowerPoint Microsoft Office PowerPoint Microsoft PowerPoint XP and Microsoft PowerPoint Download and install it in your PowerPoint environment
After installing InkTools in PowerPoint you can access InkTools through the ShowHide OfficeOne InkTools menu item introduced in the rightclick menu during the slide show
