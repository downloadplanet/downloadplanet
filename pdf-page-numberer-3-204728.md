# PDF Page Numberer 3
PDF Page Numberer is a Acrobat plug-in tool for Acrobat which is used to automatically page number your documents, you can put customizable page numbers anywhere on the page, with any font size, font name & font type also, features include saving settings for later use for other pdf's, page start & end ranges, start and stop page counts for page groups + many other options.
NOTE: This Software requires Full Version of Acrobat 5 or greater by Adobe Software
