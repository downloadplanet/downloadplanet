# Shortcut Manager for Visio
Shortcut Manager for Visio enables you to
Define your own keyboard shortcuts to ribbon buttons and menu items
Define shortcuts to commands that arent available directly through the Visio User Interface
Shortcut Manager for Visio works with Microsoft Visio bit Microsoft Office Visio and Microsoft Office Visio Download the addin and install it in your Visio environment After the addin is loaded successfully you can start assigning shortcuts using the customize shortcuts dialog The customize shortcuts dialog can be invoked by clicking Customize Shortcuts button in AddIns ribbon tab in Microsoft Visio or Tools Customize Shortcuts menu item in Microsoft Office Visio and Microsoft Office Visio
