# Shortcut Manager for Visio
Shortcut Manager for Visio enables you to:
Define your own keyboard shortcuts to ribbon buttons and menu items.
Define shortcuts to commands that aren't available directly through the Visio User Interface.
Shortcut Manager for Visio works with Microsoft Visio 2010 (32-bit), Microsoft Office Visio 2007 and Microsoft Office Visio 2003. Download the add-in and install it in your Visio environment. After the add-in is loaded successfully, you can start assigning shortcuts using the customize shortcuts dialog. The customize shortcuts dialog can be invoked by clicking Customize Shortcuts button in Add-Ins ribbon tab in Microsoft Visio 2010 or Tools | Customize Shortcuts... menu item in Microsoft Office Visio 2007 and Microsoft Office Visio 2003.
