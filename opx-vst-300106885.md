# OP-X VST
The OP-X is an OB-X VST emulation with separate signal path for each voice, SEM-Filter emulation and comprehensive detuning options for rich and organic sound. The OP-X soundlibrary contains all the classic sounds from Van Halen, Europe, Rush, Alan Parsons, Prince and many more.
