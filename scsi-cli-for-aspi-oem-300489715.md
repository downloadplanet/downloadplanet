# SCSI CLI for ASPI OEM<br />The SCSI CLI for ASPI OEM software package allows the user to execute the SCSI commands via the ASPI and is intended to be used and redistributed only in the built-in form as a native part of the other solutions. All executable modules are implemented as a console applications that accept the number of arguments in the command line. The command line arguments allow user to specify the SCSI device and the SCSI command parameters. It is possible to rescan the SCSI bus of the host bus adapter and to enumerate the SCSI devices.
