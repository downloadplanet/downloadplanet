# TCP/IP logger ActiveX
TCP/IP logger ActiveX is an ActiveX component that uses Advanced TCP/IP Data Logger to get either data from an TCP/IP or UDP port or already processed and converted data besides the standard export tools in Advanced TCP/IP Data Logger.

TCP/IP logger ActiveX can be used to extend functionality of Advanced TCP/IP Data Logger and/or as one more integration method of Advanced TCP/IP Data Logger to your hardware or software application. It is an optimal way for reading data from an TCP/IP or UDP port to Microsoft Office, MatLab and LabVIEW.
