# OPC logger ActiveX
OPC logger ActiveX is an ActiveX component that uses Advanced OPC Data Logger to get either data from an OPC server or already processed and converted data besides the standard export tools in Advanced OPC Data Logger.

OPC logger ActiveX can be used to extend functionality of Advanced OPC Data Logger and/or as one more integration method of Advanced OPC Data Logger to your hardware or software application. It is an optimal way for reading data from an OPC server to Microsoft Office, MatLab and LabVIEW.
