# vMaps for PowerPoint
![vMaps for PowerPoint](https://mycommerce.akamaized.net/api/pimages/P300486695/BIG/300486695.PNG)
This addin creates a single new 'Vector Maps' group and button in the Insert tab of the PowerPoint ribbon. The button provides access to more than 200 county vector maps and 3000+ USA county maps that can be easily inserted into a PowerPoint slide with a user selectable style and then edited in PowerPoint. User styles include 'Use selected shape', 'Use Theme', 'Apply a Grid', 'Fill with a flag'. Selected maps support editable place labels and multi-level geography. For example, the USA map can be inserted as the outline line, a group of all states or a single state. The powerful Auto Color feature allows you to recolor the map with data from an Excel file.
