# OBJ mesh to solid
OBJ_mesh_to_solid is a stand-alone application that helps user blend NURBS surfaces from OBJ polygon mesh. It can import .obj file and export NURBS surface to .igs file. When you have any polygon mesh created from CG software ( e.g Maya) and would like to blend high quality NURBS surfaces, this small software may help you.
